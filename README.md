# ✈️ Aeromiles

Aeromiles adalah proyek sistem loyalty program maskapai yang dikembangkan untuk memenuhi tugas **TK3 Basis Data**. Sistem ini memungkinkan pengguna untuk mengumpulkan, mengelola, dan menggunakan poin aeromiles secara efisien.

---

## Deskripsi Proyek
Aeromiles dirancang untuk mensimulasikan bagaimana program frequent flyer bekerja. Sistem ini mencakup:
- Pengelolaan data pengguna
- Akumulasi poin dari transaksi penerbangan
- Penukaran poin menjadi reward
- Monitoring aktivitas pengguna

---

## Tujuan
- Menerapkan konsep **basis data relasional**
- Melatih perancangan **ERD & skema database**
- Mengimplementasikan query SQL yang kompleks
- Mensimulasikan sistem nyata dalam dunia industri

---

## Teknologi yang Digunakan
- SQL
- PostgreSQL / MySQL
- DBMS Tools (DBeaver / pgAdmin)
